# asana-search - read what's on the plan

Pull the relevant Asana task or project into the conversation before you reason about status.

## When to use

- The operator asks "what's due this week?", "who owns this?", "what's left in this project?".
- You are about to add work and need to avoid duplicating an existing task.

## Steps

1. Search Asana for the task, project, or assignee with the Asana tools; prefer recent activity.
2. Read the task fully - assignee, due date, status, comments - before summarizing.
3. If several tasks match, list them and ask which one rather than guessing.
4. Link the task you used so the operator can open it.

## Rules

- Reading Asana is safe and runs freely; **creating or changing** tasks waits for approval -
  never assume permission to assign, reschedule, or complete work.
