# notion-write-page - create or update a Notion page

Turn a request into a well-structured Notion page, proposed for the operator's approval.

## When to use

- The operator says "put this in Notion", "make a page for…", or "update the … doc".

## Steps

1. Search first (see notion-search) to avoid duplicating an existing page - update it if it exists.
2. Draft the page with a clear title, a one-line summary at top, then the body in short sections.
3. Propose the create/update; it is an outward change, so it surfaces as an approval card - wait.
4. After approval, share the page link back and note what you changed.

## Rules

- Prefer updating the canonical page over creating near-duplicates.
- Keep the operator's voice; don't invent facts to fill a template.
