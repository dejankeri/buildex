# notion-search - find the source of truth in Notion

Pull the relevant Notion page or database entry into the conversation before you reason about it.

## When to use

- The operator asks about something that likely lives in Notion (a doc, a project, a client record).
- You are about to write or change something and need the current version first.

## Steps

1. Search Notion for the topic using the Notion tools; prefer the most recently edited match.
2. Open the top hit and read it fully before summarizing - do not answer from the title alone.
3. If several pages match, list them briefly and ask which one, rather than guessing.
4. Quote or link the page you used so the operator can verify.

## Rules

- Reading Notion is safe and runs freely; **creating or editing** a page waits for the operator's
  approval - never assume permission to write back.
