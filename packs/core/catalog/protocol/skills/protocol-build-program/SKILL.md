# protocol-build-program — draft a program the coach can assign

Propose a training/nutrition change grounded in the client's actual record.

## When to use

- The operator says "build … a program", "adjust …'s plan", or "update …'s nutrition".

## Steps

1. Look up the client first (see protocol-client-lookup) so the plan fits where they actually are.
2. Draft the program with realistic, practical numbers — sensible sets/reps, round portions.
3. Present it for the coach to review; **assigning or messaging the client waits for approval**.
4. After approval, assign it and confirm what the client will see.

## Rules

- Land on clean, usable numbers over hitting a target with awkward fractions.
- Mirror the coach's existing style; never assign without the coach's approval.
