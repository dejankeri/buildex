# calendly-availability - read the schedule

Pull the relevant Calendly event types, booked events, or availability before you reason about times.

## When to use

- The operator asks "who's booked this week?", "when am I free?", "what event types do I have?".
- You are about to send a scheduling link and need the right event type first.

## Steps

1. List event types, scheduled events, or availability with the Calendly tools.
2. Read the details - invitee, time, event type, timezone - before summarizing.
3. If several event types fit, present them and ask which the operator wants to share.
4. Note the exact event-type slug so the link step is unambiguous.

## Rules

- Reading the schedule is safe and runs freely; **creating a scheduling link, or cancelling/rescheduling**
  a booked event waits for the operator's approval - never move someone's meeting unattended.
