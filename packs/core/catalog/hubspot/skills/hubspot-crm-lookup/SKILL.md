# hubspot-crm-lookup - read the CRM before you act

Pull the relevant HubSpot contact, company, or deal into the conversation before you reason.

## When to use

- The operator asks "who is this?", "what stage is this deal?", "when did we last talk to them?".
- You are about to log activity or write outreach and need the current record first.

## Steps

1. Search HubSpot for the contact, company, or deal with the HubSpot tools.
2. Read the record - lifecycle stage, deal amount, last activity - before summarizing.
3. If several records match, list them and ask which rather than merging in your head.
4. Link the record so the operator can verify.

## Rules

- Reading the CRM is safe and runs freely; **writing back** - logging activity, moving a deal stage,
  editing a contact - waits for the operator's approval. Never assume permission to change the record.
