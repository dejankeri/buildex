# gmail-draft - write an email the operator can send

Draft a clear, on-voice email and hand it over for review before anything is sent.

## When to use

- The operator says "reply to …", "email … about …", or picks a message from triage to answer.

## Steps

1. Read the thread you're replying to so the draft actually fits the context.
2. Write a short, direct draft: the ask or answer up front, then any detail, then a clear close.
3. Present the draft for the operator to edit; **sending is an outward action and waits for approval**.
4. Only send after explicit approval; confirm what went out.

## Rules

- Match the operator's tone; don't over-formalize a casual thread.
- Never send without approval, and never invent commitments on the operator's behalf.
