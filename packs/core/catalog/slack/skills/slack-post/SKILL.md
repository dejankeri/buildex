# slack-post - draft a Slack message for approval

Write a Slack-appropriate message and hand it over before it posts.

## When to use

- The operator says "post in #…", "reply to … in Slack", or "send the team an update about…".

## Steps

1. Check the channel/thread context so the message fits the room.
2. Draft it short and skimmable - Slack, not email; lead with the point.
3. Confirm the target channel/thread and present the draft; **posting waits for approval**.
4. Post only after approval; confirm where it went.

## Rules

- Match the channel's tone (a standup update reads differently than a client escalation).
- Never post without approval; never @-here/@-channel unless the operator asked.
