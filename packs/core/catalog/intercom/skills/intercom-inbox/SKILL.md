# intercom-inbox - read the support conversation

Pull the relevant Intercom conversation or contact into the conversation before you reason.

## When to use

- The operator asks "what did this customer ask?", "what's open?", "have we replied to…?".
- You are about to draft a reply and need the full thread first.

## Steps

1. Search Intercom for the conversation or contact with the Intercom tools.
2. Read the whole thread - the customer's ask, prior replies, and status - before summarizing.
3. If several conversations match, list them and ask which rather than guessing.
4. Quote the relevant message so the operator can verify context.

## Rules

- Reading the inbox is safe and runs freely; **replying, snoozing, or closing** waits for the
  operator's approval - never send a message to a customer without it.
