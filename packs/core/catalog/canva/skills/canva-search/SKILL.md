# canva-search - find the design before you touch it

Pull the relevant Canva design or asset into the conversation before you reason about it.

## When to use

- The operator asks "do we have a design for…?", "find the … template", "export the … post".
- You are about to create something and want to reuse an existing brand asset first.

## Steps

1. Search Canva for the design, folder, or asset with the Canva tools; prefer recent edits.
2. Open the top match and read its structure before summarizing or proposing edits.
3. If several designs match, list them and ask which one rather than guessing.
4. Link or note the design you used so the operator can open it.

## Rules

- Reading and exporting existing designs is low-risk; **creating or editing** a design waits for
  the operator's approval - never overwrite brand work without it.
