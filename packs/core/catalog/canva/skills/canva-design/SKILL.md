# canva-design - create or edit a Canva design

Turn a request into a concrete Canva design, proposed for the operator's approval.

## When to use

- The operator says "make an Instagram post for…", "update the deck", "design a graphic that…".

## Steps

1. Search first (see canva-search) to reuse a template or brand asset instead of starting blank.
2. State what you'll create or change: format, key copy, and which template or brand kit you'll use.
3. Propose the create/edit; it is an outward change, so it surfaces as an approval card - wait.
4. After approval, share the design link and the export the operator asked for.

## Rules

- Prefer editing on-brand templates over inventing a new look each time.
- Don't put unverified claims or fake stats into marketing copy; keep it to what the operator gave.
